LPI2C Master Receive Block

This block allows the user to perform a blocking or non-blocking master receive operation (request data from slave).

Block Image

Inputs:

  • Address - set slave address here (UINT8 data type).
  • SendStop - should be true or false, depending on wheather you want to send stop at the end of the transfer or not (eg.: use a constant with the value of "true" if you want to send some values and stop) (BOOLEAN data type)
  • Length - set the size (in bytes) of the data you want to request from slave.

Outputs:

  • Data - data received from the slave (UINT8 data type).

Parameters and Dialog Box

The block dialog consists of the following tabs:

  • General

General


Instance

Select an instance of LPI2C to use (on S32K144 there is only one).

Transfer mode

Select blocking or non-blocking transfer operation.

Timeout

Insert timeout value (in ms) for the transfer.

Note: Only available if Blocking transfer mode is selected.

10-bit address

Check the box if you want 10-bit addressing; leave it unchecked if you want 7-bit.
