LPI2C Master Disable/Enable Block

The main functionality of the block is to enable/disable I2C interrupts.

Block Image

Inputs:

  • None

Outputs:

  • None

Parameters and Dialog Box

The block dialog consists of the following tabs:

  • General

General


Instance

Select an instance of LPI2C to use (on S32K144 there is only one).

Transmit Data

Enable/disable TDIE.

Receive Data

Enable/disable RDIE.

End Packet

Enable/disable EPIE.

STOP Detect

Enable/disable SDIE.

NACK Detect

Enable/disable NDIE.

Arbitration Lost

Enable/disable ALIE.

FIFO Error

Enable/disable FEIE.
