FreeMaster Configuration Block

This block allows the user to configure the FreeMaster embedded-side software driver, which implements the serial interface between the application and the host PC.
FreeMASTER is a PC-based application serving as a real-time monitor, visualization tool, and a graphical control panel of embedded applications based on Freescale/NXP Semiconductor processing units.

Block Image

Inputs:

  • None

Outputs:

  • None

Parameters and Dialog Box

The block dialog consists of the following tabs:

  • General
  • Advanced

General

The General tab contains the following parameters:

Interface

Selects communication interface.

Baud Rate

Available speed selections for the serial baud rate in kbps.

RxD pin

For usability RxD Pin selection is displayed as: (PortPinName: [Signal name | Description]) format.

TxD pin

For usability TxD Pin selection is displayed as: (PortPinName: [Signal name | Description]) format.

Mode

Selects interrupt or poll-driven serial communication.

  • Long Interrupt - complete message processing in interrupt.
  • Short Interrupt - SCI FIFO-queuing done in interrupt.
  • Poll-driven - no interrupt needed, polling only

ISR Priority

Interrupt priority level.

Show advanced options

Shows or hides the Advanced options.

Advanced

The Advanced tab contains the following parameters:

Use Scope

Enables/Disables scope support.

Max Variables ( for oscilloscope support)

Maximum number of scope variables (2..8).

Use Recorder

Enables/Disables recorder support.

Max Variables (for recorder support)

Maximum number of recorder variables (2..8).

Buffer size (0 - 65536)

User-allocated recorder buffer.

Timebase(0 - 16384)

Recorder time base specifies how often the recorder is called in the user application.

Automatic buffer size

  • Checked - buffer size is set to 0.
  • Unchecked - buffer size can be set manually in Buffer size edit box.

Buffer size (1 - 255)

Communication buffer size.

Default FIFO receiver queue size

  • Checked - the default receive FIFO queue size is set to 0.
  • Unchecked - the receive FIFO queue size can be set manually in FIFO receiver queue size edit box.

FIFO receiver queue size (1 - 255)

Receive FIFO queue size is used only with Short Interrupt.
