Read Register Block

This block allows the user to read data from a specified register.

Block Image

Inputs:

  • None

Outputs:

  • Data from a specified register.

Parameters and Dialog Box

The block has the following parameters:

Peripheral

Available peripheral selections for the device specified in Main Configuration block.

Register

Available register selections with read-only or read-write access rights for the selected peripheral.

Register description

Short description of the selected register.

Address

The address of the register from which the data will be read. The value is displayed in gray out for user information.

Size (bits)

The expected size of data that will be read from the selected register.

Block Dependency

Please do the following:
  • Configure the Target Configuration block.

Block Miscellaneous Details

  • None
