Write Register Block

This block allows the user to write 8/16/32 bits value to a selectable register.

Block Image

Inputs:

  • Data to be written in a specified register.

Outputs:

  • None

Parameters and Dialog Box

The block has the following parameters:

Peripheral

Available peripheral selections for the device specified in Main Configuration block.

Register

Available register names selections with write-only or read-write access rights for the selected peripheral.

Register description

Short description of the selected register.

Address

The address of the register in which the input data will be written. The value is displayed only for user information and it cannot be modified.

Size

The size of the accepted data to be written in the expected register.

Block Dependency

Please do the following:
  • Configure the Target Configuration block.

  • None
