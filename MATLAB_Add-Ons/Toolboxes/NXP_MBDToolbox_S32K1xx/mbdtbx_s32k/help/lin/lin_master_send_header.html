LIN Send Header Block

The main functionality of the block is transmitting a LIN Header on the LIN bus.

Block Image

Inputs:

  • ID - from this ID, the PID (protected ID) will be automatically calculated, and put on the LIN bus.

Outputs:

  • None.

Parameters and Dialog Box

LIN

Selects which LIN (LPUART) module to use.

Block Dependency

  • Use LIN_Config Block to configure LIN.

Block Miscellaneous Details:

  • None
