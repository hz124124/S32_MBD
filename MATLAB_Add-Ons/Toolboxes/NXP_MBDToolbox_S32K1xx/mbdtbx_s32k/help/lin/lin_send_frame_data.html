LIN Send Frame Data Block

This block allows the user to send data over the LIN bus.

Block Image

Inputs:

  • Data - The data to be transmitted over the LIN bus.

Outputs:

  • None.

Parameters and Dialog Box

LIN

Selects which LIN (LPUART) module to use.

Block Dependency

  • Use LIN_Config Block to configure LIN.

Block Miscellaneous Details:

  • None
