Digital Input ISR Block

The main functionality of the block is to process GPIO ISRs.

Block Image

Inputs:

  • None

Outputs:

  • Function-Call

Parameters and Dialog Box

Input Pin

Selects which Pin to trigger ISR. For usability the Input Pin selection is displayed as: (Port Name : [Function : Description]) format.

Interrupt Configuration

Sets up the GPIO interrupt requests:

  • when logic 0
  • on rising-edge
  • on falling-edge
  • on either-edge
  • when logic 1

ISR Level( 0 - 15)

Interrupt priority level
