FTM PWM Enable Disable Block

The main functionality of the block is to allow the user to Enable/Disable FTM PWM pulses on output pins.

Block Image

Inputs:

  • None

Outputs:

  • None

Parameters and Dialog Box

Module

Select which FTM module to use.

  • Range: 0 – FTMn

Action

  • Enable/Disable

Block Dependency

FTM PWM Config Block must to be configured.

Block Miscellaneous Details

  • None
