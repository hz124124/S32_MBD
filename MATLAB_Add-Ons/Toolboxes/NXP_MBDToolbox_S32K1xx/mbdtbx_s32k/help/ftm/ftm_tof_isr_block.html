FTM Timer Overflow Block

The main functionality of the block is to generate a function call on a timer overflow.

Block Image

Inputs:

  • None

Outputs:

  • Function-call

Parameters and Dialog Box

Module

Select which FTM module to use.

  • Range: 0 – 3

ISR Priority

Interrupt priority level. A lower priority value indicates a higher ISR priority

  • 0 – 3 - S32K11x
  • 0 – 15 - S32K14x
