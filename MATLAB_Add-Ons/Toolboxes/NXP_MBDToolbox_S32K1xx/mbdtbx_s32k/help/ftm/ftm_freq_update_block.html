PWM Frequency Update Block

The functionality of the block is to update the PWM Frequency for FTM during runtime.

Block Image

Inputs:

  • Frequency (uint32)

Outputs:

  • None

Parameters and Dialog Box

Module

Select which FTM module to use.

  • Range: 0 – FTMn

PWM Period expressed in units of

Selects the units in which FTM will update the frequency.

  • Ticks
  • Hz

Block Dependency

FTM PWM Config Block must to be configured.

Block Miscellaneous Details:

  1. None.
