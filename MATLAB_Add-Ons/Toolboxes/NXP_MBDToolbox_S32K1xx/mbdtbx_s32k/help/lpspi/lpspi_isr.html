LPSPI ISR Block

This block allows the user to trigger a subsystem upon a specific event (END OF TRANSFER).

Block Image

Inputs:

  • None.

Outputs:

  • Trigger - function call for a subsystem.

Parameters and Dialog Box

The block dialog consists of the following tabs:

  • General

General


SPI

Select an instance of LPSPI to use.

Role

Select the role (either Master or Slave).

Events

Event checkboxes (depending on selected Mode).

    Master:
  •   Transfer END
    Slave:
  •   Transfer END
