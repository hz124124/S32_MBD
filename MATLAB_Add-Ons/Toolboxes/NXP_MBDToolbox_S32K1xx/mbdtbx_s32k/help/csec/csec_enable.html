CSEc Enable Block

This block allows to enable the CSEc module.

Note:This is only available for S32K14x parts (no support for S32K11x).

Block Image

Inputs:

  • None

Outputs:

  • None

Parameters and Dialog Box

User keys

The number of user keys to allocate in EEEPROM for the CSEc module.
