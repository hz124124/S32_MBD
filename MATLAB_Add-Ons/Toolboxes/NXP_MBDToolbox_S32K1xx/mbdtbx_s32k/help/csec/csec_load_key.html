CSEc Load Key Block

This block allows to load a key.

Note:This is only available for S32K14x parts (no support for S32K11x).

Block Image

Inputs:

When loading an encrypted key:
  • M1 (uint8[16])
  • M2 (uint8[32])
  • M3 (uint8(16)
When loading a plain key:
  • Plain key (uint8(16))

Outputs:

  • Status (status_t)

Parameters and Dialog Box

Key ID

The ID of the updated key slot: Master ECU key, RAM key, User Keys 1 to 17.
