CSEc Decrypt Block

This block allows to decrypt a cipher text.

Note:This is only available for S32K14x parts (no support for S32K11x).

Block Image

Inputs:

  • Cipher text (uint8_t*) - the length must be a multiple of 16.
Available only for Cipher Block Chaining (CBC) mode:
  • IV (uint8_t[16]) - initialization vector.

Outputs:

  • Status (status_t)
  • Plain text (uint8_t*)

Parameters and Dialog Box

Mode

The AES decryption mode: Electronic Codebook (ECB) or Cipher Block Chaining (CBC).

Key ID

The key used for decryption: RAM key or one of the User Keys 1 to 17.

Timeout

Timeout in milliseconds as a 32 bits unsigned integer. In case the operation times out the returned status is STATUS_TIMEOUT.
