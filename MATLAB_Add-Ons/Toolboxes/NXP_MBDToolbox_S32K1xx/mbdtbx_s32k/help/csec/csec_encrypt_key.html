CSEc Encrypt Key Block

This block allows to encrypt the key before loading it into CSEc module, by computing the M1, M2, and M3 values.

Note:This is only available for S32K14x parts (no support for S32K11x).

Block Image

Inputs:

  • None

Outputs:

  • Status (status_t)
  • M1 (uint8[16])
  • M2 (uint8[32])
  • M3 (uint8(16)

Parameters and Dialog Box

Key ID

The ID of key that will be encypted.

Key

The key that will be encrypted.

Auth key ID

The ID of authorization key needed to encrypt the key.

Auth key

The authorization key in plain needed to encrypt the key.

Counter

Increase this counter when you want to replace the key.
