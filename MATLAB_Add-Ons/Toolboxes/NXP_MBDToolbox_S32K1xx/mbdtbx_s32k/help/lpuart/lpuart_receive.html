LPUART Receive Block

The main functionality of the block is reading data from a LPUART module.

This block will start a non-blocking receive over UART. If you want to trigger a subsistem after the end of the transfer, see the Tx/Rx ISR block.

Block Image

Inputs:

  • None

Outputs:

  • Data (uint8)

Parameters and Dialog Box

LPUART Module

Selects which LPUART module to use.

Block Dependency

  • Use LPUART_Config Block to configure LPUART.
