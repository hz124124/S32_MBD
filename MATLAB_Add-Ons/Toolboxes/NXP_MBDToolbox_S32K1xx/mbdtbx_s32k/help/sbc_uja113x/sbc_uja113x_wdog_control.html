SBC UJA113x Wdog Control Block

This block allows the user to reset the watchdog timer, to change the watchdog mode or its nominal period.

Block Image

Inputs:

  • None

Outputs:

  • None

Parameters and Dialog Box

Reset Watchdog

It resets the watchdog timer.

Mode Control

It allows the user to change the operating mode of the watchdog.

Nominal period

It allows the user to change the nominal period of the watchdog.

The watchdog mode and watchdog period can only be changed when the SBC is in Standby mode.

Block Dependency

Please do the following:
  • Configure the UJA113x Configuration block.
  • Configure the LPSPI Configuration block.
