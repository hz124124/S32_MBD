FLEXIO I2C MASTER Receive ISR Block

This block allows the user to perform a non-blocking master receive operation.

Block Image

Inputs:

  • Address - set slave address here (UINT8 data type).
  • Length - Set receive size, in bytes (UINT32 data type).

Outputs:

  • Trigger - the user can trigger a subsystem at the end of transfer.
  • Data - received data from the slave (UINT8 vector data type).

    • Note: You receive a number of bytes equal to Length (input).

Parameters and Dialog Box

The block dialog consists of the following tabs:

  • General

General


Instance

Select an instance of flexio to use (must be linked with the Config block).

Note: The Config block linked to this block must have Interrupts mode selected.
