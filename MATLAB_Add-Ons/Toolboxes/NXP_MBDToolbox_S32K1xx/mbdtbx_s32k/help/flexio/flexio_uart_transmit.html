FLEXIO UART Transmit Block

This block allows the user to perform a blocking transmit operation.

Block Image

Inputs:

  • Data - vector that needs to be sent (UINT8 data type).

Outputs:

  • None

Parameters and Dialog Box

The block dialog consists of the following tabs:

  • General

General


Instance

Select an instance of flexio to use (must be linked with the Config block).

Note: The Config block linked to this block must have Polling mode selected.

Timeout

Insert timeout value (in ms) for the transfer.

Block Dependency

  • FlexIO Configuration Block
