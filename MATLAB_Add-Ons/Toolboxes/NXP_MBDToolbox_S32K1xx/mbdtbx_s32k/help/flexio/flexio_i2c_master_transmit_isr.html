FLEXIO I2C MASTER Transmit ISR Block

This block allows the user to perform a non-blocking master transmit operation.

Block Image

Inputs:

  • Data - vector that needs to be sent to the slave (UINT8 data type).
  • SendStop - should be true or false, depending on wheather you want to send stop at the end of the transfer or not (eg.: use a constant with the value of "true" if you want to send some values and stop) (BOOLEAN data type)
  • Address - set slave address here (UINT8 data type).

Outputs:

  • Trigger - the user can trigger a subsystem at the end of transfer.

Parameters and Dialog Box

The block dialog consists of the following tabs:

  • General

General


Instance

Select an instance of flexio to use (must be linked with the Config block).
