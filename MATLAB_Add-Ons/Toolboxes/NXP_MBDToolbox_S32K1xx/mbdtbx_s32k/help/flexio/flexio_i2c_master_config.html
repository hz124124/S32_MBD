FLEXIO I2C MASTER Configuration Block

This block allows the user to configure the FlexIO module to emulate I2C communication (master mode).

The FlexIO module can run (only) as master and you can select the baudrate and which pins to be used (from the 8 flexio pins available on S32kxx).

Block Image

Inputs:

  • None

Outputs:

  • None

Parameters and Dialog Box

The block dialog consists of the following tabs:

  • General
  • Pins

General


Instance

Select an instance of flexio to use (must be linked with Tx/Rx blocks).

Mode

Select either Polling or Interrupts mode, depending on what kind of Tx/Rx block you want to use, as follows:

  • Transmit / Receive Blocks (blocking) - select POLLING
  • Transmit ISR / Receive ISR Blocks (non-blocking) - select INTERRUPTS

Baud Rate

Select the baud rate for FlexIO (in Hz).

Note: Max Value for baud rate can be FlexIO Clock / 6 .

Pins

Pins

Select the flexio pins to be used from the dropdown list.
