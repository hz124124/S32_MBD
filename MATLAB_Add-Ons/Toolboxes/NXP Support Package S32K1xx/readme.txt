This package contains NXP Support Package for S32K1xx v.2.1.0 which is the 
step-by-step installer for the NXP Model-Based Desing Toolbox for S32K1xx and 
the additional software and tools needed to simulate, build and deploy of 
embedded applications to NXP S32K1xx Automotive Microprocessors platforms

To run the installer invoke NXP_Support_Package_S32K1xx in MATLAB Command Window

For support, please use this link: https://community.nxp.com/community/mbdt
